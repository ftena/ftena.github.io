Creación automática de Personajes

Para empezar debo deciros que esta utilidad NO es mia. Es sacada de una página web: El Calabozo. Podeis visitar esta página o escribir a sergio@calabozo.com, diciéndole si os gusta este generador o no.

Este generador sirve a los Dungeon Master para crear PNJ´s fundamentalmente, de este modo se ahorra tiempo. Puedes crear personajes desde el nivel 1 al 13. Para ver como se hace debeis disponer de un browser que soporte JAVA.

Ahora vereis el programa, pero si lo quieres bajar para probarlo por ti mismo o para ponerlo en tu Web ve a la página de Calabozo donde se explica detalladamente, pulsa aqui para ir alli :-)
